\# Анализатор дискового пространства (Disk Analyzer)



Программа для автоматической проверки жестких дисков компьютера. Она показывает, сколько места занято, и предупреждает, если диск переполнен.



\## Для чего это нужно?

Если вы системный администратор или просто следите за сервером, вам нужно знать, когда место на диске заканчивается. Эта программа:

1\. Сканирует все диски.

2\. Считает общее и занятое место.

3\. Выводит предупреждение, если диск заполнен более чем на \*\*90%\*\* (этот порог можно менять).



\## Как пользоваться программой



\### Шаг 1. Подготовка

Убедитесь, что на компьютере установлена \*\*Java\*\* (версии 17 или новее).



\### Шаг 2. Настройка (необязательно)

Программа имеет файл настроек `src/main/resources/app.properties`. Вы можете открыть его любым текстовым редактором (Блокнот, Notepad++).



Параметры:

\* `usage.threshold.percent=90` — При каком проценте заполнения бить тревогу. Если поставите 50, программа покажет все диски, которые заполнены наполовину.

\* `output.units=GB` — В чем измерять объем (GB - гигабайты, MB - мегабайты, TB - терабайты).



\### Шаг 3. Запуск

1\. Откройте командную строку (терминал) в папке с программой.

2\. Соберите программу одной командой:

&nbsp;  \* \*\*Windows\*\*: `./gradlew build`

&nbsp;  \* \*\*Linux/Mac\*\*: `./gradlew build`

3\. Запустите программу:

&nbsp;  ```bash

&nbsp;  java -jar build/libs/disk-analyzer-1.0.0.jar



\### Что вы увидите (Результат)



Программа выведет отчет на экран в формате JSON (текстовый формат данных).

Пример:

{

&nbsp; "threshold\_exceeded": \[

&nbsp;   {

&nbsp;     "filesystem": "C:",     <-- Имя диска

&nbsp;     "total": "500GB",       <-- Всего места

&nbsp;     "used\_percent": 95      <-- Занято 95% (Внимание! Больше порога)

&nbsp;   }

&nbsp; ],

&nbsp; "summary": {

&nbsp;   "total\_gb": 1000,         <-- Общий объем всех дисков

&nbsp;   "total\_used\_gb": 850      <-- Сколько всего занято

&nbsp; }

}
