# Disk Analyzer (Вариант 4)

Консольная программа для проверки свободного места на дисках.

## ⚠️ ВАЖНО: Требования перед запуском
Для работы **обязательно** нужна **Java 17** или **Java 21**.
**Не используйте Java 24 (Beta)** — с ней программа не соберется.

Проверить свою версию:
```bash
java -version
